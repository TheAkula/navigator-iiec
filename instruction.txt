Для добавления ссылок на файлы нужно:

1) добавить ссылку в json файл(*1), находящийся в папке client/build/data в виде

{
    "path": [], // "https://domain.com" - для ссылок на другой сайт или файл, ["stud2"] - для ссылок на папку с файлами
    "title": "", // название ссылки
    "role" : 1 // 1 - ссылка на папку с файлами, 2 - ссылка на другой сайт или файл
    // "img": "images/image.svg" // - путь до картинки (опционально), должна находиться в папке client/build/images

}

2) если ссылка ведет до папки с файлами нужно добавить ее в файл files.json в папке server/dist/data в виде

{
  "path": "smk$/", // локальный путь до папки из коренной папки (192.168.0.102) 
  "roles": [ // типы ролей которые будут иметь доступ к ней
    {
      "role": 0, // тип роли: 0 - роль преподавателя, 1 - роль студента, 2 - роль админа
      "rights": [0] // типы прав работы с файлами: 1 - чтение, 2 - запись
    },
    {
      "role": 2,
      "rights": [0, 1]
    }
  ]
}

*1 - для вкладки "Главная" файл main.json
     для вкладки "Заявки" файл requests.json
     для вкладки "Студенту" файл student.json
     для блока "МЕНЮ" файл left-sidebar.json
     для блока "ДОПОЛНИТЕЛЬНО" файл right-sidebar.json
